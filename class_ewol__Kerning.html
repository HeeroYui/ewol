class: ewol::Kerning

Description:

Kerning properties of one specific Glyph with an other
Without Kerning : 
                                    
       \          /      /\         
        \        /      /  \        
         \      /      /    \       
          \    /      /
           \  /      /        \     
            \/      /          \    
       v          v a          a

With Kerning : 
                                    
       \          /  /\             
        \        /  /  \            
         \      /  /    \           
          \    /  /
           \  /  /        \         
            \/  /          \        
       v        a v        a

Note: The "Kerning" is the methode to provide a better display for some string like the "VA" has 2 letter that overlap themself. This name Kerning
